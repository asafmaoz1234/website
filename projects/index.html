Serverless website

AWS S3 / CloudFront / Route53 / HTML / CSS

What? Highly scalable, cost-effective, low latency website. Why? I wanted to run my own website, own and set up the infrastructure and reduce monthly costs of hosting.
How?
S3 bucket - configured for "static website hosting" holds the files. S3 is a very low cost, highly scalable and low latency data storage.
Cloudfront distribution (CDN) - for accelerated content delivery, its also cost-effective, and has default security for DDoS attacks.
Route53 - A reliable and cost-effective DNS service
AWS certificate manager - to issue an SSL certificate for my domain.
Resulting in a highly scalable low latency website that costs less than 1$ a month.
github repository (html-css only)



Do it yourself irrigation

Arduino / C++ / DIY

What? Automated irrigation without a water connection.
Why? I did not have a water connection near my pepper plants, and understanding how important is regulated watering, I needed an automated way to water my pepper plants even when I forget or out of town.
How? I used Arduino Uno, which is an amazing low-cost microcontroller, wrote a simple program to schedule activation and connected the arduino to a 12V water pump and the pump to ~40 liter of water (5 days of watering during summer).
With components that cost less the 50$, I got an automated irrigation anywhere I need.
short video (it only looks like a bomb)
github repository for the schedule program



SeaVision Pro

data fetcher

JAVA / AWS Lambda / reactive / CloudWatch / CI-CD

What? A service to notify me when sea visibility should be great.
Why? Instead of actively checking for consecutive days with low waves, which indicates visibility is good enough for snorkeling I want to get notified on it.
How? AWS Lambda - Query a weather service api for measurements in specific geolocation coordinates and according to a set of rules determine if it should notify me.
SQS - get the notifications events and trigger a different lambda that handles emails and notifications.
CloudWatch - scheduled event that triggers the lambda once a day.
github actions - for CI-CD and testing on pull requests.
github repository



1st Israel Hot Peppers group

Social / Hot peppers / Community

What? A facebook community with over 12K members for Israeli pepper heads
Why? Back in 2015, I started gardening as a hobby and found a world of hot peppers. There was no dedicated local community for sharing seeds and growing tips or products in the local market that were "Super hot".
How? I founded the first Israeli hot peppers group which today has over 12K members, we share guides, ideas and help each other grow some of the hottest peppers in the world.
We had several meetings and today there are several hot peppers communities, even a book was written on growing super hots, and you can find allot more super hot products.
