AWS-Powered Serverless Website Supercharged, Secure, and Surprisingly Affordable!

AWS S3 / CloudFront / Route53 / HTML / CSS / CI-CD

Embarking on the journey of creating my own website was a decision fueled by curiosity and a commitment to efficiency.

For this project, I decided to craft a serverless website hosted on AWS.

The content is stored on S3 and supercharged by CloudFront. Route53 is used for DNS management and ACM used for SSL certificate.

For updating content, I wrote an efficient CI/CD process that automates deployment of content to AWS S3 and ensures a seamless experience by automatically clearing the CDN cache.

This dynamic stack not only supercharges content delivery but also comes with a surprisingly budget-friendly price tag—less than 1$ a month.

Do it yourself irrigation Budget-Friendly Automated Irrigation System!

Arduino / C++ / DIY

An automated irrigation system in a water-challenged environment—a creation that ensures your plants stay hydrated, regardless of a direct water connection.

Using components like the Arduino Uno (an amazing low-cost microcontroller), a 12V water pump, and a 40-liter water container, this system offers a lifeline of regulated hydration for up to 5 days, so you won't forget to water your super hot pepper plants and can leave home without a worry.

All for a remarkably affordable price of less than $50.

SeaVision Pro Data Collector

JAVA / AWS Lambda / reactive / CloudWatch / CI-CD

The Data collector module within SeaVision Pro is a sophisticated component designed to provide SeaVision Pro with the latest weather information for selected coastal locations. Its primary goal is to fetch and store accurate and current weather data.

The module runs as an AWS Lambda, triggered by a CloudWatch event, and data is securely stored in our aws hosted postgres database.
The projects Uses Spring Boot, reactive core and Java 17.
GitHub Actions are utilized for CI/CD and running tests on pull requests.
Dockerized for local development.

Effortless Deployment with GitHub Actions Workflows

github action / CI-CD / js / git / workflow

No more hassles of manually updating SeaVision Pro AWS Lambda code or uploading website files to S3 and invalidating the distribution.
I wanted to experience GitHub Actions firsthand, so I set up the following workflows:

  • Sync your S3 bucket with modified files and invalidates the CloudFront distribution automatically after a merged pull request to the main branch.
  • Effortlessly deploy a JAR file from the target folder to an AWS Lambda function.
  • Handle everything from building, testing, and packaging to utilizing marketplace actions for updating the Lambda on a merged pull request to the main branch.

Seamless, efficient, and safe—GitHub Actions have never been this empowering!
