Arduino / C++ / DIY

Do it yourself irrigation

Why? I needed an automated way to water my pepper plants, but I did not have a water connection. So I connected an Arduino Uno to a 12V water pump and ~40 liter of water.
using a simple program I wrote and components that cost less the 50$, I am now able to set up an automated irrigation anywhere I need.

JAVA / AWS Lambda / SQS / CloudWatch

Its beach time!

Why? I want to get notified when sea visibility is good enough for snorkeling in specific areas I want to dive.
So I set up a cloud watch event to everyday trigger a lambda, the lambda runs a java project that queries stormglass.io for last 3 days weather and according to a set of rules publishes a message to SQS to my email notifications queue.
github repository



Social / Hot peppers / Community

Israel Hot Peppers group

Why? Back in 2015, I started gardening as a hobby and found a world of hot peppers without a local community or products in the local market.
I founded the first Israeli hot peppers group which today has over 12K members, the group shares guides, ideas and help each other. We had several meetings and today there are several hot peppers groups, even a book was written, and you can find allot more super hot products.
